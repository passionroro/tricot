# Tricot

## Install dependencies

On macOS:

```sh
brew install opencv cmake
```

On Linux (Ubuntu):

```sh
sudo apt-get update
sudo apt-get install libopencv-dev cmake
```

## Build and run

```sh
./build.sh && ./build/tricot
```
